# meta data extractor

simple python tool to grab all exif data from a pic, shows basic info, dumps everything hidden, and if theres gps it calculates the location and opens google maps in your browser.

### requirements
- python 3
- pillow (pip install pillow)

### how to use
1. save the script as like `meta.py` or whatever
2. run it: `python meta.py`
3. when it asks "image path:" paste the full path to your image (like C:\Users\you\Downloads\pic.jpg)
4. watch it print all the data and pop maps if location is there
